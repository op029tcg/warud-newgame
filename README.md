# STAR DRIFTER 🚀

ブラウザで動く縦スクロール・シューティングゲーム。
HTML5 Canvas + バニラ JavaScript 製で、ライブラリ・ビルド不要。

## 遊び方

`index.html` をブラウザで開くだけ。

または簡易サーバーで:

```bash
# Python があれば
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 操作

| 操作 | キー / マウス |
| --- | --- |
| 移動 | マウス移動 / 矢印キー・WASD |
| ショット | 自動発射（マウス押しっぱなしで連射強化） |
| ボム | スペースキー / 右クリック |

## 特徴

- 4 種類の敵（ドローン / ウィーバー / シューター / タンク）
- パワーアップ：**P**（ショット強化）/ **B**（ボム）/ **1UP**（残機）
- スコアに応じてレベルアップし、難易度が上昇
- ボム発動で画面の敵と敵弾を一掃
- WebAudio による効果音、パーティクル、画面シェイク演出
- ハイスコアは `localStorage` に保存

## ファイル構成

```
index.html   画面・HUD・各種オーバーレイ
style.css    スタイル
game.js      ゲームロジック（入力・更新・描画・サウンド）
```
