# TRION ARENA 🔷 — ランク戦シミュレーター

『ワールドトリガー』の世界観をモチーフにした、ブラウザで動く**見下ろし型アリーナ戦闘ゲーム**。
界境防衛機関「ボーダー」の隊員として、トリオン体を駆使し、襲来する近界民（ネイバー）を殲滅する。
HTML5 Canvas + バニラ JavaScript 製、ライブラリ・ビルド不要。

## 遊び方

`index.html` をブラウザで開くだけ。または簡易サーバーで:

```bash
python3 -m http.server 8000   # → http://localhost:8000
```

## 操作

| 操作 | キー / マウス |
| --- | --- |
| 移動 | WASD / 矢印キー |
| 照準 | マウス |
| 攻撃・射撃 | 左クリック |
| トリガー切替 | `1` `2` `3` `4` |
| シールド（防御） | 右クリック / Shift（長押し） |
| グラスホッパー（回避ダッシュ） | Space |

## トリガー（武器）

原作の「トリガー」を再現した4種を切り替えて戦う。発動には**トリオン**を消費する。

| キー | トリガー | ポジション | 特徴 |
| --- | --- | --- | --- |
| 1 | **弧月** | アタッカー | 正面を薙ぐ近接斬撃。高威力・低燃費・短射程 |
| 2 | **アステロイド** | ガンナー | 照準方向への高速連射。バランス型 |
| 3 | **アイビス** | スナイパー | 長押しでチャージ→離して貫通狙撃。最大威力 |
| 4 | **メテオラ** | ガンナー | 着弾で爆発する範囲攻撃。群れに有効・高燃費 |

- **シールド**：正面からの攻撃を無効化（トリオンを消費）。バッグワーム同様の防御運用。
- **グラスホッパー**：短距離ダッシュ。発動中は無敵フレームあり。

## ゲームの流れ

- **ウェーブ制**。各ウェーブで近界民・トリオン兵が襲来し、全滅させると次のウェーブへ。
- ウェーブクリアでスコアボーナス＆トリオン体を少し回復。
- **5の倍数のウェーブで大型トリオン兵（ボス）が出現**。
- トリオン体（HP）が尽きると**ベイルアウト（緊急脱出）**＝ゲームオーバー。
- スコアに応じて **C → B → A → S** のランクが上がる。ハイスコアは `localStorage` に保存。

## 登場する近界民

| 敵 | タイプ | 行動 |
| --- | --- | --- |
| マーモッド | 小型・高速 | 突進して接近戦を仕掛ける |
| バムスター | 中型・射撃 | 距離を保ちトリオン弾を撃つ |
| ラービット | 大型・重装 | 鈍足だが高耐久、接近して薙ぎ払う |
| 大型トリオン兵 | ボス | 放射弾幕・狙い撃ち・突進を使い分ける |
